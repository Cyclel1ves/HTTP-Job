# HTTP-сервер списка дел (без фреймворков)

Сервер на стандартной библиотеке Python для управления задачами.

## Запуск

```bash
python server.py 127.0.0.1 8080
```

Все запросы, которые изменяют список задач, сохраняют данные в `tasks.txt`.
При старте, если файл существует, данные восстанавливаются из него.

## API

### Создание задачи

POST `/tasks`

Тело (JSON):

```json
{"title":"Gym","priority":"low"}
```

Ответ:

```json
{"title":"Gym","priority":"low","isDone":false,"id":1}
```

### Получение списка задач

GET `/tasks`

Ответ: JSON-массив задач.

### Отметка о выполнении

POST `/tasks/{id}/complete`

Ответ:
- 200 и пустое тело, если задача найдена
- 404, если задачи нет
